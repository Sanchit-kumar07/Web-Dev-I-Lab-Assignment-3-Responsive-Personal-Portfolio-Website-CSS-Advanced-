# Web-Dev-I-Lab-Assignment-3-Responsive-Personal-Portfolio-Website-CSS-Advanced-
Web Dev I Lab Assignment 3
Project Title
Responsive Personal Portfolio Website (CSS Advanced)
Description
This project involves the design and build of a one-page responsive portfolio website. The website incorporates modern CSS techniques, including Flexbox, CSS Grid, media queries, and animations, to create a polished and interactive layout that adapts seamlessly to various screen sizes.
Objectives
The main objective is to design and build a functional and responsive one-page portfolio using the specified CSS techniques for a professional appearance across devices.
